VuePress

Vue-powered Static Site Generator

What is VuePress?

  • Minimalistic static site generator with a Vue-powered theming system and Plugin API
  • A default theme optimized for writing technical documentation
  • JAMSTACK(JavaScript,Api,Markdown)
  • Written By the Vue Core Team!
  • Each page generated by VuePress has its own pre-rendered static HTML.
  • Vue takes over the static content and turns it into a full Single-Page Application (SPA).
  • A VuePress site is in fact a SPA powered by Vue, Vue Router and webpack.
  • Easy to host to static services like Surge and Netlify and to integrate tools like Netlify CMS

What Can I Use?

  • Can use Markdown Files (Includes built in Markdown Extensions)
  • Any Markdown file that contains a YAML frontmatter block will be processed
  • Can use Vue components that are registered in project.(Don't have to learn VueJS to get started!)

Default Theme

  • Vuepress Default Theme comes with customizable sidebar,search box, and Navbar
  • Navbar automatically searches all headers with markdown
  • Writing your own themes using Vue Components

Plugins

  • Plugins generally add global-level functionality to VuePress.
  • Examples: Back to the top, Google Analytics, PWA Support
